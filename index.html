Aplikasi POS (Point of Sale) UD. Rusdianto Komputer Kudus

When : 2017 - 2018
Contribution : Full Stack Developer
Description :
Pada kegiatan ini berhasil membangun aplikasi sistem pengelolaan stok barang dan transaksi pada UD. Rusdianto Komputer berbasis Web dengan menggunakan metode pengembangan SDLC (System Development Life Cycle). Sistem informasi tersebut dirancang menggunakan pemodelan UML, dengan bahasa pemrograman PHP dan database MySQL. Dengan adanya aplikasi ini pihak instansi dapat mengelola data yang ada didalam toko. Seperti, data stok barang, data supplier, data karyawan, transaksi penjualan serta laporan dari hasil penjualan. Sehingga dapat meningkatkan produktivitas dan kinerja di toko.
Source : https://github.com/agusismawan/rusdiantokom.git

(SI-KUBA) Sistem Peminjaman Inkubator Bayi di Yayasan Bayi Prematur Indonesia Jakarta

When : 2019
Contribution : Full Stack Developer
Description :
Project ini menghasilkan sebuah sistem informasi peminjaman inkubator pada Yayasan Bayi Prematur Indonesia yang digunakan untuk mengelola dan memantau peminjaman inkubator. Sistem ini memiliki fitur responsif terhadap perangkat yang mengakses serta memiliki fitur SMS autoreply. Sistem ini berguna untuk mengelola data inkubator dan agen relawan, sehingga agen relawan dapat mengetahui informasi mengenai agen relawan yang lain. Sistem ini berguna untuk memilih agen relawan yang lokasinya terdekat dengan peminjam menggunakan fitur Google Geocoding.
Source : http://si-kuba.com
